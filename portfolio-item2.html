Ecommerce
WebApp

PHP & MYSQL

A project about ecommerce. I worked on this project for about 4 months. Languages I used for this ecommerce project: HTML5, CSS3, Javascript, AJAX, PHP & MySQL. Responsiveness only for tablets & desktop. Basically this project consists of 3 roles: 1) Administrator, 2) Help Agent and 3) Client. Full CRUD operations functionality for each one of these, except for the payment. Admin manages help agents, clients and tracks the products, help agent can chat with clients, meanwhile clients can update their profile, add products to Wishlist and ShoppingCart and can track his products.
