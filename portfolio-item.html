Job Portal

Java & MYSQL

So, it took me about 2 months to complete this web project on backend, while I used a free template on front end(though I had to make a few html templates myself). So this project consists of two roles: Administrator and User(companies). CRUD mode for both roles, Admin manages users and posts, whereas user(company) can make posts (job offers), update and delete them, also can update company details.
Technologies used for backend: Java, MySQL, JSP, Servlets, MVC Model.
